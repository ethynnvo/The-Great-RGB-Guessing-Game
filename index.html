The Great
RGB
Guessing Game

The goal of the game is simple: to guess the correct RGB color. RGB represents red, green and blue within a scale of 0 - 255. Therefore rgb(255, 0, 0) is red, rgb(0, 255, 0) is green and rgb(0, 0, 255) is blue. You must now guess which square is the correct color based on the rgb code given.
